# ダッシュボード用 GitHub API 接続許可

## 概要

- 日時: 2025-11-10
- 担当: GPT-5 Codex (ツンデレ先輩)
- 目的: ダッシュボードが GitHub コミット履歴を取得する際に発生していた CSP 違反を解消する

## 症状

- `/dashboard` へアクセスするとブラウザが `connect-src` の CSP 違反を大量に報告
- `https://api.github.com/.../commits` へのフェッチがブロックされ、ダッシュボード側で外部データを取得できない状態

## 対応内容

- `lib/utils/csp.ts` の `connect-src` に `https://api.github.com` を追加し、GitHub REST API への接続を許可
- ドキュメントログ (本ファイル) を追加して対応内容を記録

## 影響範囲

- ダッシュボードおよび GitHub API を利用する UI のネットワークリクエスト
- 同一 CSP を継承する全ページ (GitHub API を利用する場合は許可済みのため追加対応不要)

## 今後のメモ

- 追加の外部サービスへ接続する場合は `lib/utils/csp.ts` を更新し、ドキュメントへ記録する運用を維持すること
- フロントエンドから直接外部 API を叩かず、必要に応じて自前の API 経由に切り替える検討も継続する
