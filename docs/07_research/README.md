# 調査・研究

技術調査、競合調査、パフォーマンス調査などのレポートを管理するディレクトリです。

## ディレクトリ構造

```
07_research/
├── 2025_10/
│   ├── 20251010/
│   │   └── [調査レポート].md
│   └── 20251015/
│       └── [調査レポート].md
└── 2025_11/
    └── ...
```

## 調査レポートに含めるべき内容

1. **調査概要**: 何を、なぜ調査したか
2. **調査結果**: 発見したこと
3. **メリット・デメリット**: 良い点・悪い点
4. **推奨事項**: 調査結果を踏まえた提案
5. **参考資料**: 参照した URL、ドキュメント

## テンプレート

`/docs/templates/research-template.md` を使用してください。
