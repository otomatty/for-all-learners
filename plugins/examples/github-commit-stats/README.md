# GitHubコミット統計プラグイン

GitHubコミット行数をカレンダーに表示し、ダッシュボードに統計を表示するプラグインです。

## 機能

- **カレンダー拡張**: カレンダーセルにGitHubコミット行数のバッジを表示
- **Widget表示**: ダッシュボードに今月のGitHubコミット統計を表示
- **詳細情報**: カレンダーの詳細パネルにコミット統計を表示

## 設定

プラグインを有効化する前に、以下の設定が必要です：

1. **GitHub認証トークン**: `github_oauth_token` をプラグインストレージに保存
   - トークンには `repo` スコープが必要です

2. **プラグイン設定**:
   - `owner`: GitHubリポジトリのオーナー名
   - `repo`: GitHubリポジトリ名

## 使用方法

1. プラグインをインストール
2. 設定画面で `owner` と `repo` を入力
3. GitHub認証トークンを設定
4. プラグインを有効化

## 表示内容

### カレンダーセル

- **バッジ**: コミット行数（+42, -10 など）またはコミット数
- **色**: 
  - 緑: 追加行数が多い
  - 赤: 削除行数が多い
  - 青: コミットのみ（行数変化なし）

### ダッシュボードWidget

- 今月の合計コミット行数
- コミット件数
- 追加・削除行数の内訳

### 詳細パネル

- コミット数
- 追加行数
- 削除行数
- 純増行数

## 開発

このプラグインは、F.A.Lプラグインシステムの拡張機能を使用しています：

- **Calendar Extension**: カレンダーUIにデータを表示
- **UI Extension**: ダッシュボードにWidgetを表示
- **Integration Extension**: GitHub APIを呼び出し
