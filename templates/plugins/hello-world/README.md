## 開発

```bash
# 依存関係をインストール
bun install

# ビルド
bun run build

# ウォッチモードでビルド
bun run watch

# テスト
bun run test
```

## 使用方法

1. プラグインをビルド
2. F.A.Lアプリケーションでプラグインをインストール
3. プラグインを有効化
4. コマンドパレットから "Hello World" コマンドを実行
